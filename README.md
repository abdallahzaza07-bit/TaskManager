task manager project:

this project is a java task manager application that allows users to add and remove tasks using a graphical user interface.
the program was built using the java swing library for the gui and an arraylist for storing tasks.
github was used for version control to track progress and manage changes during development.

Prerequisites and requirments:

1.Java jdk 17 or later installed.
2.A java supported ide such as visual studio code, or a terminal with java configured.
3.A computer capable of running java applications.

How to run the program:

1.open the project folder in an ide such as  visual studio code.
2.ensure java is installed and properly configured.
3.run the TaskManager.java file.
4.A graphical window will open, allowing the user to interact with the application.
